# Kontakt Library Manager

Windows 桌面应用，管理 Kontakt 8 音色库（BobDule 版本）。

## 功能特性

- **音色库管理**：添加、移除、批量入库 Kontakt 音色库
- **非标准库支持**：管理没有 .nicnt 文件的第三方采样库
- **分类标签**：多标签分类，自定义分类管理
- **音色搜索**：实时搜索过滤音色库
- **注册信息查看**：查看注册表、XML、JSON 来源位置
- **暗色主题**：VS Code 风格暗色 UI

## 系统要求

- Windows 10/11 64-bit
- 需要管理员权限（用于注册表操作）

## 安装

下载 [Releases](../../releases) 页面的安装包，运行安装程序。

或从源码运行：

```bash
pip install -r requirements.txt
python main.pyw
```

## 使用说明

### 标准库管理

1. 点击"管理库文件夹" → 添加标准库目录
2. 目录下有 .nicnt 文件且已注册的音色库会自动显示
3. 使用"添加音色库"或"批量入库"注册新的音色库

### 非标准库管理

1. 点击"管理库文件夹" → 添加非标准库目录
2. 目录下的所有子文件夹会自动作为非标准库显示
3. 非标准库独立管理，不参与 Kontakt 注册
