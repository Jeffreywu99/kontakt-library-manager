# Kontakt Library Manager

Windows 桌面应用，管理 Kontakt 音色库。

## 兼容性说明

### 支持的 Kontakt 版本

| 版本 | 支持状态 | 备注 |
|------|---------|------|
| Kontakt 8 | ✅ 已测试 | 主要目标版本 |
| Kontakt 7 | ✅ 已测试 | 完全兼容 |
| Kontakt 6 | ⚠️ 应该兼容 | 未实际测试，注册表位置相同 |
| Kontakt 5 及以下 | ❓ 未知 | 未测试 |

### 支持的用户群体

| 用户类型 | 入库功能 | 管理功能 | 备注 |
|---------|---------|---------|------|
| **BobDule 版** | ✅ 完全支持 | ✅ 支持 | 主要目标用户 |
| **官方正版** | ❌ 不需要 | ✅ 支持 | 正版库通过 Native Access 自动授权 |
| **R2R 版** | ❓ 可能不兼容 | ❓ 未知 | 授权机制可能不同 |

> **重要提示**：
> - 正版用户不需要使用入库功能，库会通过 Native Access 自动显示
> - 正版用户可以使用本软件的分类、备注等管理功能
> - 如果正版用户误操作入库，软件会自动阻止重复写入

## 功能特性

- **音色库管理**：添加、移除、批量入库 Kontakt 音色库
- **非标准库支持**：管理没有 .nicnt 文件的第三方采样库
- **分类标签**：多标签分类，自定义分类管理
- **音色搜索**：实时搜索过滤音色库
- **注册信息查看**：查看注册表、XML、JSON 来源位置
- **暗色主题**：VS Code 风格暗色 UI

## 系统要求

- Windows 10/11 64-bit
- 需要管理员权限（用于注册表操作）

## 安装

下载 [Releases](../../releases) 页面的安装包，运行安装程序。

或从源码运行：

```bash
pip install -r requirements.txt
python main.pyw
```

## 使用说明

### 标准库管理

1. 点击"管理库文件夹" → 添加标准库目录
2. 添加目录时，程序会扫描未入库的库并询问是否入库
3. 目录下有 .nicnt 文件且已注册的音色库会自动显示
4. 使用"添加音色库"或"批量入库"注册新的音色库
5. 移除目录时，可选择是否同时从 Kontakt 中移除

### 非标准库管理

1. 点击"管理库文件夹" → 添加非标准库目录
2. 目录下的所有子文件夹会自动作为非标准库显示
3. 非标准库独立管理，不参与 Kontakt 注册
4. 移除目录只影响软件显示，不会删除文件
