# Release Notes

## v0.4.1

### 修复
- **批量入库勾选框点击偏移**：修复点击区域不在勾选框位置的问题
- **批量入库误入库保护**：已入库的库默认不勾选，放在列表底部，有分隔线标识

### 改进
- **添加标准库目录交互**：添加目录时自动扫描未入库的库并询问是否入库
- **移除标准库目录交互**：移除前显示已入库库的数量，询问是否同时从 Kontakt 移除
- **管理员权限请求**：启动时自动请求管理员权限（UAC）

### 兼容性说明
- 支持 Kontakt 6/7/8（K6 未实际测试）
- 主要面向 BobDule 版用户
- 正版用户不需要入库功能，可使用管理功能

### 下载
- `KontaktLibraryManager-Setup-0.4.1.exe` - Windows 安装包
- `KontaktLibraryManager.exe` - 便携版（需自行构建安装包）

---

## v0.4.0

### 新功能
- **非标准库目录**：在"管理库文件夹"中可添加非标准库目录，自动扫描子文件夹
- **音色库搜索**：工具栏搜索框，实时过滤
- **左侧分类面板**：新增"标准库"和"非标准库"两个默认分类

### 改进
- **扫描逻辑重构**：完全基于用户添加的目录显示内容
  - 标准库目录：只显示有 .nicnt 且已注册的库
  - 非标准库目录：自动扫描所有子文件夹
  - 不在用户目录中的库不再显示（彻底过滤 SWAM、UAD 等）
- **UI 重设计**：VS Code 风格暗色主题
- **管理库文件夹对话框**：分为标准库目录和非标准库目录两个列表

### 下载
- `KontaktLibraryManager-Setup-0.4.0.exe` - Windows 安装包

## v0.3.0

### 改进
- **架构重构**："入库即显示，删除即消失"
- 主列表只显示已注册的库（有注册表/XML/JSON 记录）
- 入库时自动把父目录加入库文件夹列表

### 修复
- 入库"可见但不可用"问题：使用 .nicnt 中的 RegKey 作为注册表键名
