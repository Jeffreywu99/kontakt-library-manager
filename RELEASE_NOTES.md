# Release Notes

## v0.5.0

### 新功能
- **首次启动自动发现**：首次打开软件自动扫描注册表，显示所有已注册的 Kontakt 音色库，无需手动配置库文件夹
- **增量扫描**：后续启动只对比注册表变化（新增/移除），毫秒级完成
- **精准识别**：基于 `Visibility=3` + `HU` + `JDX` 三要素规则识别 Kontakt 音色库，零误判
- **强制完整扫描**：刷新按钮右键菜单可强制从注册表重新发现

### 改进
- **扫描逻辑重构**：从"文件夹驱动扫描"改为"注册表驱动发现"
- **扫描缓存**：首次完整扫描后缓存结果，后续启动秒开
- **状态栏显示扫描信息**：库数量和上次扫描时间
- **启动加载提示**：启动时显示"正在自动发现音色库..."

### 修复
- 不再误显示 Battery 4 Factory Library、Massive X Factory Library 等非 Kontakt 库
- 通过 ContentDir 去重，解决同一库多个注册表键名的问题
- 自动排除 Arturia、UAD、NKS 预设等非 Kontakt 条目

### 下载
- `KontaktLibraryManager-Setup-0.5.0.exe` - Windows 安装包

## v0.4.1

### 修复
- **批量入库勾选框点击偏移**：修复点击区域不在勾选框位置的问题
- **批量入库误入库保护**：已入库的库默认不勾选，放在列表底部，有分隔线标识

### 改进
- **添加标准库目录交互**：添加目录时自动扫描未入库的库并询问是否入库
- **移除标准库目录交互**：移除前显示已入库库的数量，询问是否同时从 Kontakt 移除
- **管理员权限请求**：启动时自动请求管理员权限（UAC）

### 兼容性说明
- 支持 Kontakt 6/7/8（K6 未实际测试）
- 主要面向 BobDule 版用户
- 正版用户不需要入库功能，可使用管理功能

### 下载
- `KontaktLibraryManager-Setup-0.4.1.exe` - Windows 安装包
- `KontaktLibraryManager.exe` - 便携版（需自行构建安装包）

## v0.4.0

### 新功能
- **非标准库目录**：在"管理库文件夹"中可添加非标准库目录，自动扫描子文件夹
- **音色库搜索**：工具栏搜索框，实时过滤
- **左侧分类面板**：新增"标准库"和"非标准库"两个默认分类

### 改进
- **扫描逻辑重构**：完全基于用户添加的目录显示内容
  - 标准库目录：只显示有 .nicnt 且已注册的库
  - 非标准库目录：自动扫描所有子文件夹
  - 不在用户目录中的库不再显示（彻底过滤 SWAM、UAD 等）
- **UI 重设计**：VS Code 风格暗色主题
- **管理库文件夹对话框**：分为标准库目录和非标准库目录两个列表

### 下载
- `KontaktLibraryManager-Setup-0.4.0.exe` - Windows 安装包
