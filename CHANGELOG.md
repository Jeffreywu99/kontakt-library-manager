# Changelog

## [0.2.0] - 2026-05-20

### Added
- 刷新按钮：重新扫描所有库文件夹
- 文件夹大小显示（MB / GB 自动切换）
- UI 防抖优化：快速点击不再卡顿，备注输入延迟保存

### Changed
- 大幅简化：移除注册库显示、清理残留、打开注册表等冗余功能
- 只显示用户添加的文件夹中的库（标准 + 非标准）
- 左侧面板简化为：全部 / 标准库 / 非标准库 / 用户分类 / 未分类 / 隐藏

### Fixed
- 注册表操作覆盖 HKCU 路径
- 批量入库对话框
- 备注保存性能优化

## [0.1.0] - 2026-05-19

首个可用版本。
